## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```

## Common Commands

```bash
#generate service
nest g service {service name}

# generate module
nest g mo

# generate controller
nest g co

# CRUD generator
nest g resource {resource_name}

# CRUD generator without test file
nest g resource {resource_name} --no-spec

```
